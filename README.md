# 🎬 无限剧本杀 (Infinite Scenario)

> 由 OpenClaw AI 自动生成的无限流小说游戏，每天一个新世界，永远猜不到下一个副本！

---

## 🎮 游戏简介

**无限剧本杀**是一款自动更新的点击式文字冒险游戏，基于无限流小说的概念。

- **每日更新**：AI 每天生成一个全新的剧本副本
- **能力继承**：完成副本后获得技能，在后续世界中发挥
- **随机世界**：丧尸末日、修仙武侠、科幻赛博、谍战悬疑...
- **多结局设计**：每个副本都有多个结局和隐藏剧情

## 🎯 游戏机制

### 核心玩法

1. **选择副本**：从剧本列表中选择一个世界进入
2. **阅读场景**：沉浸式体验 AI 生成的剧情
3. **做出选择**：每个场景 3-5 个选择，影响剧情走向
4. **完成目标**：达成主任务获得胜利
5. **解锁技能**：完成结局获得新技能

### 能力继承系统

玩家可以保留之前剧本中获得的技能：

- **生存术**：搜索成功率 +20%，发现隐藏道具
- **领导力**：NPC 更愿意跟随
- **洞察力**：发现隐藏信息，降低死亡概率
- **谈判技巧**：避免冲突，获得额外信息
- ...（AI 持续生成新技能）
